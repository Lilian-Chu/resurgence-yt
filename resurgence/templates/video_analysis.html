Welcome to the Resurgence Video Analytics tool! You can input multiple YouTube videos and the tool will extract basic information about each video, including the title, channel name and subscribers, publication date, views, and tags. This tool is a work in progress, but we plan to add more functionality to the tool so it can better help analyze video performance.

What the Video Analytics tool can do now:

  • Input a playlist of videos and extract basic info about the videos in the playlist

What the Video Analytics tool will be able to do soon:

  • Find trends in video stats such as common tags for videos
  • Visualize trends in data using graphs

To get started, enter a URL to a YouTube playlist of videos. The playlist must be public!
